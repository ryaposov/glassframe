## About

Glassframe is a Node.js server that allows you to proxy websites without browser restrictions such as CORS.

## Main features

* Proxy the website document and all the resources requested
* Parse all coming resources to replace the URL's and endpoints

## Built with

Glassframe is a Node.js server application build with the power of Express framework and Express HTTP Proxy.

* Express
* Express HTTP Proxy
